# Banana Roadtrip Scoring

Simple SwiftUI iOS app for tracking banana-themed roadtrip scores between riders.

## Features

- Name each road trip session
- Track up to five players
- Give each player a name and preset emoji avatar
- Add banana points or subtract peel-slip penalties one tap at a time
- Show the current leader and running trip total
- Persist the current session locally between app launches

## Open In Xcode

1. Open `BananaRoadtripScoring.xcodeproj` in Xcode.
2. Select an iPhone simulator or connected device.
3. Build and run.

## Game Theme

- Each player is a roadtripper collecting banana points.
- Use `+` for a banana win and `-` for a peel slip or correction.
- The crown highlights the current leader.
